Overview

Purpose

The purpose of this website is to help users visualize and organize their dream travel destinations in an engaging and dynamic way. The site offers inspiration and interactive features for exploring different types of travel experiences.

Audience

This site is for travel enthusiasts, students, and anyone interested in discovering new destinations. It's particularly aimed at younger audiences who enjoy visual planning and interactive tools.

Dynamic elements

The site will include an array of destination objects displayed as cards. Users will be able to filter destinations by continent or type (beach, city, nature). Each destination will include a button to “Add to My List,” which stores selections using localStorage. Badges like “Visited” or “High Priority” will display based on destination properties, using conditional logic.

Content

Home page

The home page will introduce the concept of a travel bucket list and feature a randomly generated inspirational travel quote. It will also display a highlighted destination with an image and a link to explore the full bucket list.

Bucket List

The bucket list page will include cards for each destination, with an image and type label. Users can filter destinations by category using a dropdown menu. Content is generated dynamically using JavaScript and organized via objects and arrays.
